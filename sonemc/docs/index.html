Valentine - 1.1-dev4

Developed: 1.20.2
Tested and Stable: 1.19, 1.20, 1.20.6, 1.21

All those hearts in one plugin. No, this plugin wont find girlfriend for you.

Features:

  • Can kiss/hug players with customizable cooldown
  • Leaderboard
  • Customizable in "config.yml"
  • Use of permissions
  • Heart particles around player
  • Changes "love" with "❤" with changable color, of course

Commands:

Command Alias Description Permission
/valentines /val, /vday Displays plugin's commands valentine.use
/valentinesplayer (player) /vstats (player) Displays player's stats valentine.use
/valentinesleaderboard /vlb Displays leaderboard valentine.use
/hug (player) /valstats, /vstats Hugs player valentine.use
/kiss (player) /valstats, /vstats Kisses player valentine.use

Permissions:

Permission Description
valentines.use Allows to use all commands
valentines.effect You will have hearts around you

config.yml:

Copied!
          # Prefix and no-permission
          Prefix: "&c❤&f&lValentines&c❤&r "
          NoPermissionMessage: "&cYou do not have permission to use this command."
          
          #  Messages
          Hug: "&fYou hugged &d{player} ❤"
          HugReceived: "&d{player} &fhugged you&d! ❤"
          
          Kiss: "&fYou kissed &d{player} ❤"
          KissReceived: "&d{player} &fkissed you&d! ❤"
          
          # Chat
          heart-color: "&c"
          replace-love-with-hearts: true
          
          # Effects
          effect:
            enabled: true
            particle-density: 1
            update-rate: 2
            max-height: 2.0
            start-height: 0.1
            radius: 0.85
          
          # Enabled Worlds
          enabled-worlds:
            - world
            - world_nether
          
          # Cooldowns (in seconds)
          cooldowns:
            hug: 60
            kiss: 60
