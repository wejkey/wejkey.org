NameTagIt - 1.1

Plugin that allows to put nametag on mobs using only command, so you dont need any nametags. But you need to look at the mob when you naming it.

Features:

  • Name a mob without NT
  • Remove NT from a mob

Commands:

Command Alias Description Permission
/nametagit (name) /nti (name) Assign a name to the mob you are looking at nametagit.use
/nametagitremove /ntir Removes the name nametagit.remove

Permissions:

Permission Description
nametagit.use Allows player to add NT to a mob
nametagit.remove Allows player to remove NT from a mob

config.yml:

prefix: "&bNameTagIt&f â–¸ "
no-permission: "&cYou cannot use this command"
named-message: "&aYou added nametag on this mob"
removed-message: "&cYou removed a nametag from this mob"
no-mob-message: "&cYou are not looking at mob"

Valentine - 1.0-dev6

All those hearts in one plugin. No, this plugin wont find girlfriend for you.

Features:

  • You can Kiss / Hug someone
  • You can see how many kisses and hugs you got
  • 84% of messages are editable in config.yml
  • Heart particles around player
  • Data Storing

Commands:

Command Alias Description Permission
/valentines - Displays plugin's commands valentine.use
/hug (player) - Send a hug to another player valentine.use
/kiss (player) - Send a kiss to another player valentine.use
/valentinesplayer (player) - View Valentine's stats of a player valentine.use

Permissions:

Permission Description
valentines.use Can use the plugin

config.yml:

          # Permissions
          no-permissions: "&cSadly, you have no permissions"
          
          # Hug Messages
          Hug: "&f{player} &ahugs you."
          HugReceived: "&fYou were hugged by &a{player}!"
          hugged: "&fYou hugged &a{player}"
          
          # Kiss Messages
          Kiss: "&f{player} &akissed you."
          KissReceived: "&fYou were kissed by &a{player}!"
          kissed: "&fYou kissed &a{player}"
          
          # Chat Features
          heart-color: "&c"  # Color of heart that replaces "love"
          
          # Particle Effect
          effect: true  # Enable/disable the heart spiral effect
          
          # Worlds where the effect is enabled
          enabled-worlds:
            - world
            - lobby

Team - 1.0-dev1

Displays server's active AdminTeam players.

Features:

  • Editable Prefixes, Priorities, Messages
  • Able to see who from Admins is online

Commands:

Command Alias Description Permission
/team - Displays the team list team.use
/adminteam - Displays the team list team.use
/staff - Displays the team list team.use
/supportteam - Displays the team list team.use

Permissions:

Permission Description
team.use Display the team list(s)
team.owner Put prefix "Owner" to a player in the team list
team.developer Put prefix "Developer" to a player in the team list
team.helper Put prefix "Helper" to a player in the team list
team.builder Put prefix "Builder" to a player in the team list
team.support Put prefix "Support" to a player in the team list

config.yml:

          # Prefixes
          prefixes:
            team.owner: "&cOwner"
            team.developer: "&9Developer"
            team.helper: "&aHelper"
            team.builder: "&bBuilder"
            team.support: "&7Support"
          
          # Priorities (1 = highest priority)
          priorities:
            team.owner: 1
            team.developer: 2
            team.helper: 3
            team.builder: 4
            team.support: 5
          
          # Messages
          message:
            online: "&aOnline"
            no-permission: "&cYou don't have permission for this command!"
