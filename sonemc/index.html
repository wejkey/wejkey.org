FAQs

What version of Minecraft do the plugins support?

All these plugins are focused on version 1.20 and above but some plugins are supported on older version as well.

How do I report a bug or issue?

If you encounter a bug or issue, please report it on the GitHub Issues page of the plugin. Alternatively, you can ask about it in the Discord server.

Can I use these plugins on a public server(s)?

Yup, I'd be happy to.

How often do you update the plugins?

Depends on many factors. Time, good sleep and plugin itself :D Updates can take few days but there is atleast one plugin update per week.

What does "exp" and "dev" means?

"Exp" for Experimental. That means, plugin is released because of fun. No frequent updates. Just trying to see if people likes it. "Dev" for Under Development or Development. These plugins are frequently updated from small patches to "big" changes before stable version.
