About SoneMC

SoneMC was officially launched on February 17, 2025, as a small but passionate project focused on creating custom Minecraft plugins. These plugins are designed for versions 1.20 and above, though they may work with lower versions as well. Thank you for using them! FAQs

Are the plugins compatible with all versions?

No. These plugins are built for versions above 1.20. But some of them may work on lower versions. Depends if I tested them on lower versions or not. Supported versions are under each plugin's name in DOCS.

What are 'dev' versions?

Plugins are under development and they may not work as the stable ones. 'Dev' updates are more frequent and it can take few days/weeks until the stable version is out.

What if I found bug?

You can report the bug on Github Issues page or on Discord server.
